COMO COMPILAR:
1-Baixe todos os arquivos do repositório;
2-Abra o terminal e siga até o diretório onde os arquivos foram baixados;
3-Utilize o comando 'make'(sem aspas) para gerar arquivos .o e o exercutável;
4-Ainda no terminal, siga até o diretório 'bin'(sem aspas) e nele, encontrará o exercutável 'Programa';
5-Utilize o seguinte comando para executar o programa no terminal: './Programa'(sem aspas);
6-Utilize o comando 'make clena'(sem aspas) para excluir o executável e arquivos.o;
